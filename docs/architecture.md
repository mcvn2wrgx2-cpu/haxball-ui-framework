# Architecture

The framework is divided into multiple layers.

| Layer | Responsibility |
|---------|---------|
| Bootstrap | Framework initialization |
| Registry | Window tracking |
| Instance | Window lifecycle |
| Style | Theme management |
| Content | DOM rendering |
| Interaction | Drag & resize |
| Safety | Validation and cleanup |

## Core Goals

- Lightweight
- Modular
- Extensible
- HaxBall-focused
